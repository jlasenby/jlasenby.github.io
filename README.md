# Lasenby.com

Personal digital garden built with Jekyll and Obsidian.

## Features

- Brutalist/minimalist design inspired by Derek Sivers
- Jekyll-powered static site
- Obsidian-friendly markdown format
- GitHub Pages hosting
- Custom domain: lasenby.com

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

Visit http://localhost:4000

## Publishing Workflow

1. Write posts in Obsidian in `_posts/` folder
2. Name format: `YYYY-MM-DD-title-slug.md`
3. Include front matter (layout, title, date, permalink)
4. Commit and push via GitHub Desktop
5. Site auto-deploys to lasenby.com

## Design Philosophy

- Deep black background (#050505)
- High contrast text (#dddddd)
- Neon green links (#00ff00)
- Monospaced typography
- Single column, max-width 650px
- Mobile-first responsive design
